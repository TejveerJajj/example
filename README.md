# Python Calculator

## Description
This is a simple command-line calculator built with Python. It allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- **Addition:** Adds two numbers.
- **Subtraction:** Subtracts the second number from the first.
- **Multiplication:** Multiplies two numbers.
- **Division:** Divides the first number by the second. Includes error handling for division by zero.
- **Exponent:** Calculates the square of a number.

## How to Run
1. Make sure you have Python installed.
2. Run the `calculator.py` file from your terminal:
   ```bash
   python calculator.py
   ```
3. Follow the on-screen prompts to select an operation and enter your numbers.
